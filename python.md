# python

**Note：**

* python代码块不用{}，采用缩进的方式。

```python
if True:
	print("hello world")
else:
	print("hello python")
```

* python语句中一般以新行作为语句的结束符，但是我们可以用斜杠(\)将一行语句多行显示。

```python
total = 1 + \
		2 =* \
		3 / 5
```

* python 注释

```python
# 这是一条单行注释
print("hello world")
'''
这是一条多行注释
'''
"""
这也是一条多行注释
"""
```
